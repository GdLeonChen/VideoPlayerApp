### 2. 视频文件自动扫描

**功能描述**：
- 选择文件夹后，自动递归扫描所有子文件夹
- 自动识别视频文件（基于 MIME 类型）
- 按文件名自然排序（支持中文、数字混排）
- 排除非视频文件，减少噪音

**扫描算法**：
```java
private void scanFolderRecursive(DocumentFile folder, List<Uri> videoUris) {
    DocumentFile[] files = folder.listFiles();

    for (DocumentFile file : files) {
        if (file.isDirectory()) {
            // 递归扫描子文件夹
            scanFolderRecursive(file, videoUris);
        } else if (file.isFile()) {
            // 检查是否为视频文件
            if (isVideoMime(file.getType())) {
                videoUris.add(file.getUri());
            }
        }
    }
}
```

### 4. URI 权限持久化

```javascript
// index.html - VideoPlayer 类
loadVideoList(uris) {
    if (!uris || uris.length === 0) {
        this._showError('文件夹中没有找到视频文件');
        return;
    }

    // 从 URI 中提取文件名
    const items = uris.map(uri => {
        const uriStr = uri.url || uri;
        const fileName = uriStr.split('/').pop() || '未知视频';
        return {
            name: decodeURIComponent(fileName),
            url: uriStr,
            isBlob: false  // 直接使用 Content URI
        };
    });

    this._setPlaylist(items);
    this._playAt(0);
}
```

## 🚀 使用指南

### 如何选择文件夹

1. 启动应用，点击"选择本地视频"按钮
2. 系统打开文件夹选择器，选择包含视频的文件夹
3. 点击"允许"授予访问权限
4. 应用自动扫描文件夹内的所有视频文件
5. 自动开始播放第一个视频

### 记忆功能说明

- **首次使用**：选择文件夹后，应用会记住此文件夹
- **后续使用**：重新打开应用时，自动加载上次文件夹的视频
- **更换文件夹**：再次点击"选择本地视频"按钮，选择新文件夹覆盖记录
- **权限失效**：如果用户在系统设置中撤销权限，记忆功能会提示重新授权

## 🐛 已知问题

1. **Android 10 以下**：不支持递归扫描子文件夹，仅扫描当前层级
2. **大文件夹**：超过 1000 个视频文件的文件夹，扫描时间较长（约 5-10 秒）
3. **权限撤销**：在系统设置中撤销权限后，需要重新选择文件夹
