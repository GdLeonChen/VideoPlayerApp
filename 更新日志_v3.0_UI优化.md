## 🎨 设计令牌系统

### 色彩系统
- **主题色**：蓝色 (#3b82f6) 渐变体系
  - Primary 100：半透明背景（15% 不透明度）
  - Primary 500：主要交互色（90% 不透明度）
  - Primary Glow：发光效果（30% 不透明度）

- **语义色彩**
  - 成功：#10b981（绿色）
  - 警告：#f59e0b（橙色）
  - 错误：#ef4444（红色）

- **中性色阶**
  - Surface 0：纯黑背景
  - Surface 50/100/200/300：不同透明度的白色表面
  - Text Primary/Secondary/Tertiary：不同权重的文字颜色

### 字体系统
- **主字体**：-apple-system, PingFang SC, Helvetica Neue, Noto Sans SC, Arial
- **等宽字体**：SF Mono, Roboto Mono, Consolas
- **字号体系**：11px → 13px → 15px → 17px → 20px → 24px → 30px
- **字重**：Normal (400) → Medium (500) → Semibold (600) → Bold (700)
- **行高**：Tight (1.2) → Normal (1.5) → Relaxed (1.75)

### 间距系统
- 基于 4px 基础单位：4px → 8px → 12px → 16px → 20px → 24px → 32px → 40px → 48px

### 圆角系统
- 6px → 10px → 14px → 20px → 28px → 9999px（圆形）

### 阴影系统
- 多层次阴影：XS → SM → MD → LG → GLOW（发光效果）

### 过渡系统
- 快速：150ms cubic-bezier(0.4, 0, 0.2, 1)
- 正常：300ms cubic-bezier(0.4, 0, 0.2, 1)
- 缓慢：500ms cubic-bezier(0.4, 0, 0.2, 1)

## 🔄 技术实现

### CSS 变量系统
使用 CSS 自定义属性（`--*`）实现设计令牌系统，方便未来主题切换和全局调整。

```css
:root {
  /* 色彩 */
  --color-primary-500: rgba(59, 130, 246, 0.9);
  /* 字体 */
  --font-family-primary: -apple-system, 'PingFang SC', ...;
  /* 间距 */
  --space-4: 16px;
  /* 阴影 */
  --shadow-glow: 0 0 20px rgba(59, 130, 246, 0.4);
  /* 过渡 */
  --transition-normal: 300ms cubic-bezier(0.4, 0, 0.2, 1);
}
```

### 毛玻璃效果
使用 `backdrop-filter: blur()` 和 `background` 渐变实现现代 UI 常见的毛玻璃效果。

```css
#topBar {
  background: linear-gradient(to bottom,
    rgba(0, 0, 0, 0.85) 0%,
    rgba(0, 0, 0, 0.4) 60%,
    transparent 100%);
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px);
}
```

### 发光效果
使用 `box-shadow` 实现主题色发光效果。

```css
#progressBar {
  background: linear-gradient(90deg,
    var(--color-primary-500) 0%,
    var(--color-primary-500) 80%,
    rgba(59, 130, 246, 0.7) 100%);
  box-shadow: 0 0 8px var(--color-primary-glow);
}
```

## 🚀 性能优化

1. **动画优化**
   - 使用 transform 和 opacity 进行动画（GPU 加速）
   - 合理使用 will-change 提示浏览器优化
   - 避免布局抖动（layout thrashing）

2. **渲染优化**
   - 使用 opacity 代替 display:none 隐藏控制栏（避免重排）
   - 渐变背景使用 CSS 硬件加速

3. **触摸优化**
   - 增大触摸热区（最小 24px）
   - 使用 touch-action: none 防止默认滚动
   - 消除 300ms 点击延迟（-webkit-tap-highlight-color）
